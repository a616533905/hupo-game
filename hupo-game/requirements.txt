# 琥珀猫冒险 - 系统依赖

## Python 依赖

无第三方依赖，使用 Python 3 标准库：
- json
- http.server
- urllib.parse
- threading
- subprocess

## Node.js 依赖

无第三方依赖，使用 Node.js 标准模块：
- http
- https
- fs
- path
- child_process

## 系统依赖

| 依赖 | 版本 | 用途 | 必需 |
|------|------|------|------|
| Python | 3.6+ | AI 桥接服务 | 是 |
| Node.js | 14+ | 语音代理服务 | 是 |
| ffmpeg | 任意 | 音频格式转换 | 语音识别必需 |

## 安装指南

### Debian/Ubuntu

```bash
sudo apt update
sudo apt install -y python3 nodejs npm ffmpeg
```

### CentOS/RHEL

```bash
sudo yum install -y python3 nodejs npm ffmpeg
# 或
sudo dnf install -y python3 nodejs npm ffmpeg
```

### Fedora

```bash
sudo dnf install -y python3 nodejs npm ffmpeg
```

### macOS (Homebrew)

```bash
brew install python3 node ffmpeg
```

### Windows

1. **Python**: https://www.python.org/downloads/
2. **Node.js**: https://nodejs.org/
3. **ffmpeg**: 
   - 下载: https://ffmpeg.org/download.html
   - 或使用 winget: `winget install ffmpeg`
   - 或使用 chocolatey: `choco install ffmpeg`

## 验证安装

```bash
# 检查 Python
python3 --version

# 检查 Node.js
node --version

# 检查 ffmpeg
ffmpeg -version
```

## 可选依赖

### Ollama (本地 AI)

```bash
# Linux/macOS
curl -fsSL https://ollama.com/install.sh | sh

# 下载模型
ollama pull gemma3:270m
ollama pull qwen2.5:3b
```

### PM2 (进程管理)

```bash
npm install -g pm2

# 启动服务
pm2 start start.sh --name hupo-game

# 开机自启
pm2 startup
pm2 save
```

## 端口说明

| 端口 | 服务 | 说明 |
|------|------|------|
| 80 | API + Web | 主服务端口 |
| 85 | Voice Proxy | 语音代理端口 |

## 启动服务

### Windows

```batch
start.bat
```

### Linux/macOS

```bash
chmod +x start.sh
./start.sh
```

### 后台运行 (Linux)

```bash
# 使用 nohup
nohup ./start.sh > hupo.log 2>&1 &

# 或使用 systemd (推荐)
sudo systemctl start hupo-game
```

## 故障排除

### ffmpeg 未找到

```bash
# 检查 ffmpeg 是否在 PATH 中
which ffmpeg

# 如果未找到，添加到 PATH
export PATH=$PATH:/usr/local/bin
```

### 端口被占用

```bash
# 查看端口占用
netstat -tlnp | grep -E '80|85'

# 或
lsof -i :80
lsof -i :85

# 结束占用进程
kill -9 <PID>
```

### Node.js 版本过低

```bash
# 使用 nvm 安装新版本
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash
nvm install 18
nvm use 18
```
